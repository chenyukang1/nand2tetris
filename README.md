# Nand2tetris
依据基本原理构建现代计算机：从与非门到俄罗斯方块

Project地址 https://www.nand2tetris.org

## Project 1
实现基本逻辑门电路，了解一些数字电路内容即可完成项目，从最基本的与非门 Nand 开始

推荐实现顺序：
- 初级逻辑门
  - Not
  - And
  - Or
  - Xor
  - Mux(选择器)
  - DMux(解复用器)
- 16位
  - Not16
  - And16
  - Or16
  - Mux16
- 多路
  - Or8Way
  - Mux4Way16(16位4选1选择器)
  - Mux8Way16(16位8选1选择器)
  - DMux4Way(解4路复用)
  - DMux8Way(解8路复用)
